# 免疫联结 (Immune-Link)

![游戏截图](PLACEHOLDER_FOR_SCREENSHOT_OR_GIF.png)

**《免疫联结》是一款为 [GameJam](https://itch.io/jam/game-jam-theme) 开发的创新性2D塔防游戏，主题为“桥梁”。**

在这个独特的策略游戏中，你将深入人体的微观世界，扮演免疫系统的核心调度官。你的任务是建造特殊的“能量桥梁”，以抵御不断涌入的病毒和病菌，保护身体的核心。

## ✨ 核心玩法

在《免疫联结》中，传统的塔防概念被重新定义。**桥梁** 是你唯一的工具，它集多种功能于一身：

*   🌉 **经济命脉**: 建造桥梁，连接能量核心与资源节点，为你的防御网络提供源源不断的能量。没有桥梁，就没有经济。
*   🗼 **防御工事**: 任何一个桥梁节点都可以被升级为具有攻击能力的炮塔。你可以根据战局需要，在能量通道的任何位置建立火力点。
*   🌐 **战略路径**: 桥梁是你扩展免疫网络的唯一途径。通过策略性地铺设桥梁，你可以到达新的资源点，或是在关键位置形成交叉火力。

### 独特机制

*   **动态建造**: 像画笔一样自由地绘制你的桥梁网络，每一次落笔都是一次深思熟虑的战略部署。
*   **深度升级**: 收集资源，解锁并升级你的桥梁。你可以强化它们的攻击力、射程，或是赋予它们独特的辅助能力，如减速光环。
*   **特殊单位**:
    *   **NK细胞**: 在战场上拾取稀有的NK细胞样本，用它来激活强大的全局协议，瞬间扭转战局！
    *   **多样化的敌人**: 面对能够自我复制、加速友军、甚至“跃迁”穿过你防线的特殊病毒，你需要不断调整战术。

## 🚀 如何运行

1.  **下载/克隆项目**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/immune-link.git
    ```
2.  **打开Godot**:
    *   确保你已安装 [Godot Engine 4.x](https://godotengine.org/download/)。
    *   在项目管理器中，选择“导入”。
    *   浏览并选择克隆到本地的项目中的 `project.godot` 文件。
3.  **运行游戏**:
    *   在Godot编辑器中，点击右上角的“播放”按钮 (或按 `F5`)。
