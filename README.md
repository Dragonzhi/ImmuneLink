## ✨ 核心玩法

在《免疫联结》中，传统的塔防概念被重新定义。**桥** 是你唯一的工具，它集多种功能于一身：

* 🌉 **经济命脉**: 建造桥梁，连接资源节点，为你的防御网络提供源源不断的资源。没有桥梁，就没有经济。
* 🗼 **防御工事**: 任何一个桥梁节点都可以被升级为具有攻击能力的炮塔。你可以根据战局需要，在能量通道的任何位置建立火力点。
* 🌐 **战略路径**: 桥梁是你扩展免疫网络的唯一途径。通过策略性地铺设桥梁，你可以到达新的资源点，或是在关键位置形成交叉火力。

### 独特机制

* **动态建造**: 像画笔一样自由地绘制你的桥梁网络，每一次落笔都是一次深思熟虑的战略部署。
* **深度升级**: 收集资源，解锁并升级你的桥梁。你可以强化它们的攻击力、射程，或是赋予它们独特的辅助能力，如减速光环。
* **特殊单位**:

  * **NK细胞**: 在战场上拾取稀有的NK细胞样本，用它来激活强大的全局协议，瞬间扭转战局！
  * **多样化的敌人**: 面对能够不断生成、加速友军、甚至“跃迁”穿过你防线的DC细胞，你需要不断调整战术。

## 🚀 如何运行

1. **下载/克隆项目**:

    `bash git clone https://github.com/Dragonzhi/ImmuneLink `

2. **打开Godot**:

   * 确保你已安装 [Godot Engine 4.x](https://godotengine.org/download/)。
   * 在项目管理器中，选择“导入”。
   * 浏览并选择克隆到本地的项目中的 `project.godot` 文件。

3. **运行游戏**:

   * 在Godot编辑器中，点击右上角的“播放”按钮 (或按 `F5`)。

4. **下载发行版**:

   * 或者在本项目的发行页面下载exe和pck文件，两者在同一目录下运行exe即可游玩。

## 🎨 关卡编辑器与MOD支持

游戏内置了简单的关卡导入/导出系统，允许玩家创建和分享自己的关卡。

### 创建与导出关卡 (面向关卡设计者)

如果你想创建自己的关卡，可以按照以下步骤操作：

1.  **准备场景**: 在Godot编辑器中，打开或创建一个新的关卡场景。你可以复制一个现有的关卡场景（例如 `scenes/levels/level01/Level01.tscn`）作为模板。
2.  **编辑关卡**:
    *   **路径**: 在 `EnemySpawnPoint` 节点下，添加或修改 `Path2D` 子节点来设计敌人的行进路线。路径的识别顺序将严格按照其**节点名称的字母顺序**排序（例如 `Path01`, `Path02`...）。
    *   **预设管道**: 在场景中直接放置 `Pipe.tscn` 实例，设置好它们的类型和方向，并将它们添加到 `pipes` 组中。
    *   **波次与关卡配置**: 选中场景中的 `WaveManager` 节点，在检查器中你可以配置本关卡的初始资源、游戏时间、教程文件路径 (`.tres`) 以及每一波敌人的详细信息。
3.  **导出为 `.json`**:
    *   在你的关卡场景中，添加一个 `LevelExporter` 节点 (场景位于 `scripts/levels/LevelExporter.gd`，直接将此脚本挂载到任意节点上即可)。
    *   选中该节点，在检查器中设置 `Export Path`，指定你想要保存 `.json` 文件的位置（通常在 `levels/data/` 目录下）。
    *   勾选 `Export Now` 复选框。脚本会自动执行导出，并将关卡数据保存为 `.json` 文件。

### 安装与游玩自定义关卡 (面向玩家)

想要游玩社区制作的关卡非常简单：

1.  **找到游戏目录**: 定位到游戏的可执行文件（`.exe`）所在的文件夹。
2.  **创建 `levels` 文件夹**: 在 `.exe` 文件的旁边，创建一个名为 `levels` 的新文件夹（如果它不存在的话，游戏启动时也会自动创建）。
3.  **放置关卡文件**: 将你下载的自定义关卡 `.json` 文件直接复制到这个 `levels` 文件夹中。
4.  **启动游戏**: 正常启动游戏。在关卡选择界面，游戏会自动优先加载 `levels` 文件夹中的自定义关卡。如果自定义关卡与内置关卡同名，则会加载自定义关卡。
